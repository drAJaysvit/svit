Fixing Dependency Issues

To resolve dependency issues caused by mismatched versions of libglapi-mesa, follow these steps:

  1. Fix broken packages:

    sudo apt --fix-broken install

  2. Install the correct version of libglapi-mesa (since we previously installed the wrong version manually):

    wget http://ftp.debian.org/debian/pool/main/m/mesa/libglapi-mesa_24.0.8-1_amd64.deb
    sudo dpkg -i libglapi-mesa_24.0.8-1_amd64.deb

  3. Try installing libglut-dev manually:

    sudo apt-get install libglut-dev

  4. Finally, try installing the desired packages again:

    sudo apt-get install libglu1-mesa-dev freeglut3-dev mesa-common-dev

  5. To Compile the program

    gcc example.c -lglut -lGLU -lGL

  6. To run the program

    ./a.out
