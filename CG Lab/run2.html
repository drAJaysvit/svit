Fixing Dependency Issues

1. Download the correct versions of the required packages:

wget http://ftp.debian.org/debian/pool/main/m/mesa/libglapi-mesa_22.3.6-1+deb12u1_amd64.deb

2. Forcefully install the downloaded package:

sudo dpkg -i --force-overwrite libglapi-mesa_22.3.6-1+deb12u1_amd64.deb

3. Fix broken dependencies:

sudo apt --fix-broken install

4. Install the required packages:

sudo apt-get install libglu1-mesa-dev freeglut3-dev mesa-common-dev libglut-dev
